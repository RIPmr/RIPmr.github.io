V1.8.0工具列表


  · 主工具箱

+ [NEW]Configuration

+ Register/unregister Dialog Bar

    启用/禁用可停靠的工具窗口

+ Object Manage Tools

+ Renamer

    批量重命名选中物体

+ Attacher

    按类型(class)合并选中物体

+ Detacher

    分离子物体(elements)

+ Arranger

    按间隔整理选中物体

+ [NEW]Nested Isolator

    多层隔离显示

+ [NEW]Auto Grouper

    自动聚类选中物体，支持按组、子物体或网格合并方式聚类

+ Selection Slot

    存储当前物体选中状态到选择集，点击选择集按钮恢复选中状态

+ Random Transer

    为选中物体赋予随机位置、旋转与缩放状态

+ [Deprecated]Poly ID Tools

+ [NEW]Poly Tools

    ①按面积、邻接面数选择poly面
    ②显示选中poly顶点或面的编号

+ Poly Recreator

    清除选中模型中的错误位置信息(尤其是来自模型空间的错误信息)。当模型产生错误结果时，请尝试用此插件"重新创建"模型进行修复

+ Spline Recreator

    将选中样条曲线转换为可执行的MaxScript，同时可用于清除样条曲线中的错误位置信息

+ [NEW]Spline Outline

    将选中样条线图形顶点按法线方向推出创建轮廓线

+ [Deprecated]Spline Knot Manager

+ [NEW]Spline Tool Box

    ①样条线顶点类型管理
    ②样条线顶点选择工具
    ③样条线图形轮廓线创建
    ④按切线生成环线
    ⑤按比例打断样条线
    ⑥样条线顶点约束
    ⑦样条线信息Debug

+ [NEW]Spline Rendering Manager

    按自定义曲线控制生成的管道模型半径

+ [NEW]Clear User Defined Props

    清空自定义物体属性

+ Material Manage Tools

+ [NEW]Material Browser

    材质浏览器，批量管理所有场景中的材质

+ Show textures in viewport

    在视图中用材质贴图替换物体颜色外观，支持标准材质与Vray材质

+ [NEW]Mat Sweeper

    清除选中物体的材质

+ [NEW]Color Mixer

    颜色混合器，在两个自定义颜色之间创建过渡色

+ Animation Tools

+ Transform Manager

    锁定物体的一个或多个Transform属性

+ Attachment Holder

    为物体快捷创建附着约束

+ [NEW]Mirror Morpher

    为Morph对象应用镜像顶点位置(不改变顶点编号)

+ Spline Controller

    为样条线的每个顶点创建控制器

+ Keyframe Filter

    删除动画关键帧中的某些属性

+ Anim Repeater

    重复并延长当前物体的选中动画帧，支持增量动画

+ Anim Recorder

    将一个物体的动画记录到另一个物体

+ Anim Rotator

    旋转行走动画

+ Anim Noiser

    为物体添加噪声动画

+ Procedural Modeling Tools

+ IES Light Creator

    为Ghost Town生成的程序建筑模型添加IES射灯(Model+Vray IES Light)

+ Light Ceil Creator

    基于选中的poly面重新拓扑生成带灯带的屋顶，灯带类型为Vray灯光物体(Vray Mesh Light)

+ Voxel Tree Generator

    生成带生长动画的程序化体素树模型

+ [NEW]Voxel Mesh Builder

    将选中模型体素化

+ Debugger Boxes

    在随机位置生成一堆带随机网格颜色的盒子模型

+ Catenary Maker

    基于选中的样条线生成悬线

+ Rect Slicer

    将模型切块

+ [Deprecated]Geo Cutter

+ Batch Boolean

    批量模型布尔切割

+ [NEW][Demo Ver.]Face Retopology

    重拓扑选中poly面

+ Sub-Spline Connector

    按百分比位置连接选中样条线的子样条线，并生成一条新的样条线，支持位置排序

+ Mesh From Spline

    基于选中样条线的子样条线生成网格模型，支持位置排序

+ Skynet Creator

    基于深度优先搜索(DFS)生成网状结构模型，使用Point物体作为基础物体的子物体时Point即可被自动识别为延伸端口(下一个物体的生成方向)

+ Snowy Surface

    为选中物体添加积雪效果(提取法线方向为Z向的面生成新的积雪模型)

+ Pillars Generator

    生成随机外观的柱子模型

+ Cobwebs Generator

    基于选中物体随机生成蜘蛛网(基于样条线)

+ Rock Generator

    生成随机外观的岩石模型

+ Feather Generator

    羽毛模型生成器

+ [NEW]Tire Generator

    生成轮胎模型，带Vray置换与Vray材质

+ Mindflow Splines Creator

    基于选中的样条线图形生成样条线描边

+ [NEW]Kitchen Cabinet Generator

    生成橱柜模型

+ [NEW]Building Generator

    基于L-System的更加一般化的建筑生成器

+ Procedural Sci-Fi Building Gen

    程序化科幻建筑生成器

+ [NEW][Demo Ver.]Random Pipe Generator

    在两个选中的Poly面之间生成无穿插管道样条线模型

+ [NEW][Demo Ver.]Procedural Cable Generator

    程序化电缆线生成器

+ [NEW][Demo Ver.]Random Frame Generator

    基于Delaunay Triangle的程序化金属框架模型生成器

+ Procedural Animating Tools

+ Range Affector

    基于控制器生成程序化运动图形动画

+ Elevator Animator

    程序化电梯动画生成器

+ Boat Animator

    船舶摇摆动画生成器，支持四种噪声类型

+ Star Trace

    生成类似延时摄影生成的星轨动画效果，可自动生成渐变拖尾Vray材质

+ Poly Fracture

    poly网格破碎，新增基于控制器的爆炸动画生成

+ Vol.Sel To Texture

    基于体积选择生成贴图序列

+ Drop Animation

    掉落动画生成器

+ Spring

    为物体的已有动画帧增加弹性次级动画效果

+ Cat Tail

    为选中的骨骼链或样条线生成飘带动画

+ PF Baker

    烘焙PF粒子到网格动画

+ Rigging Tools

+ HU_Anim Icon Set

    动画绑定用样条线图形预设集

+ Auto Rigging For MakeHuman

    为MakeHuman(1.0.2)生成的模型自动生成高级人体绑定

+ Auto Facial Rigging Tools

    自动面部绑定

+ Auto Skin Weight

    更加便捷的蒙皮权重设置方式

+ [Deprecated]Make Stretchy Bones

+ [Deprecated]Poly ID Tools

+ [NEW]Bone Tools

    骨骼工具箱
    ①将选中骨骼转化为弹性骨骼
    ②根据百分比将选中骨骼打断为多节

+ Asset Operator Tools

+ Batch Exporter

    批量导出工具

+ Batch Loader

    批量导入工具

+ Bitmap Renamer

    重命名所有当前场景中引用的贴图

+ Instances To Proxy

    将实例物体转为Vray代理

+ [NEW]Batch MS Executer

    批量.ms/.mcr文件自动执行脚本，支持子文件夹遍历

+ [NEW]MS Encrypter

    .ms脚本快捷加密方式，生成.mse文件

+ Others

+ Quickly open common file paths

    常用max路径的快捷打开方式

+ Create Holes

+ Extrude Edges
