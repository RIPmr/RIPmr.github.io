[Anime]原创CG动画《极东之夜》

正片传送门：

>>>戳我<<<

简介：


作品历时五个月完成，主要采用3ds Max制作，ZBrush 精雕&贴图绘制，AE后期。
自制一套面部表情控制系统，主要应用 ReactionManager 和 AttributeHolder 以及maxScript。



MentalRay SSS material 皮肤材质测试：


整体基调为法国中世纪风格，原画：

今年评委组十分专业，全程技术，自己不太了解或者只是略懂的东西答辩就别拿出来说了，评委会一路问到底...最重要的是，老师们都是真的懂动画，懂游戏的，交流起来很轻松，很愉快，不需要有所顾虑。
今年闭幕式有幸上台展示，同学们的提问都一针见血，也见识了其他很多很好的作品，大家的实力每年都在提高。志愿者们也都很辛苦，很负责，表示感谢！也感谢其他给过我们帮助的人!

最后分享下用到的一个max script脚本：



它可以将选中的骨骼从根骨骼开始设置为弹性骨骼并创建约束目标

rollout MSB "makeStretchyBones"( button btn1 "makeStretchyBones" pos:[7,14] width:148 height:272 toolTip:"" on btn1 pressed  do(  pts=#() for x in selection do (  pt=point size:0.4 centermarker:false axistripod:false cross:false box:true wireColor:[200,0,0]  pt.transform=x.transform  append pts pt ) for i = 1 to (selection.count) do (  b=selection[i]  posCon=position_constraint()  lookAtCon=lookAt_constraint()  b.pos.controller=posCon  posCon.appendTarget pts[i] 100  if i < selection.count then  (   b.rotation.controller=lookAtCon   lookAtCon.appendTarget pts[i+1] 100   lookAtCon.lookat_vector_length=0   lookAtCon.upnode_world=false   lookAtCon.pickUpNode=pts[i]  ) )
